| Tool | Role | Cost |
|---|---|---|
| [Claude Code](https://claude.ai/code) | Architecture, planning, KB writing, Gemini prompt authoring | €21.42/month |

| [Android Studio](https://developer.android.com/studio) | Primary IDE — build, run, debug | Free |
| [Xcode](https://developer.apple.com/xcode/) | iOS build verification (pending) | Free (macOS) |
| [VS Code](https://code.visualstudio.com) | Blog and KB editing | Free |
| [Obsidian](https://obsidian.md) | Project notes, concept documents | Free (local vault) |
| [Affinity Designer](https://affinity.serif.com/designer) | App icon, feature graphic, store visuals | One-time purchase |
| [icon.kitchen](https://icon.kitchen) | Placeholder app icon generation | Free |
| [Google Stitch](https://stitch.withgoogle.com) | UI mockups and design exploration | Free |
| [SnagIt](https://www.techsmith.com/screen-capture.html) | Screenshots | One-time purchase |
| [Toggl Track](https://toggl.com/track/) | Time tracking | Free tier |
